# Task‑04: Cross‑Browser Testing with BrowserStack

## 🎯 Objective
The purpose of this task is to run automated Selenium tests across multiple browsers and devices using BrowserStack’s Selenium Grid. This ensures that the login/form submission functionality works consistently across Chrome, Firefox, Safari, and Edge.

## 🔍 Scope
- Automate login and form submission tests.
- Execute tests on Chrome, Firefox, Safari, and Edge.
- Verify functional consistency across desktop, tablet, and mobile.
- Document any issues or inconsistencies.

## ⚙️ Tools Used
- Selenium WebDriver
- BrowserStack Automate (Selenium Grid)
- Python + PyTest (or Java/TestNG)
- Documentation in Markdown

## 📂 Deliverables
- TEST_CASES.md → Detailed test cases with results
- docs/notes.md → Setup and approach
- docs/report.md → Findings summary
- docs/issues.md → Browser‑specific issues
- test_login.py → Selenium automation script
